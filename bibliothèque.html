Projet Bibliothèque de Vidéos YouTube

Le projet de la bibliothèque de vidéos YouTube est une tâche que nous avons reçue de notre école. L'objectif est de créer un site en HTML, CSS et JavaScript, centré sur une bibliothèque de vidéos avec laquelle les utilisateurs peuvent interagir. Nous avons terminé la première version du site en utilisant ces technologies. Dans le cadre de ce projet, nous allons le continuer plus tard dans l'année en intégrant du PHP et du MySQL afin de permettre le stockage de données et l'interaction via une page administrateur pour gérer la bibliothèque.
